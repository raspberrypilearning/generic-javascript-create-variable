```javascript
var animal = "cat";
```

```javascript
var score = 30;
```

변수를 만드려면, 변수의 이름을 만들고 값을 설정하세요. 변수의 이름은 항상 `var` 뒤에 오고, 공백이 포함되지 않도록 합니다.
